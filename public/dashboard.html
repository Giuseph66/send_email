API Documentation

Base URL

${window.location.origin}

Enviar Email

POST /send-email 
curl -X POST ${window.location.origin}/send-email \
  -H "Content-Type: application/json" \
  -d '{
    "remetente": "seu-email@gmail.com",
    "destinatario": "destino@exemplo.com",
    "subject": "Assunto do email",
    "message": "Conteúdo da mensagem"
  }'

Listar Remetentes

GET /remetentes 
curl ${window.location.origin}/remetentes

Histórico de Emails

GET /emails 
curl ${window.location.origin}/emails

Manual de Uso

1. Configuração do Gmail
  • Ative a verificação em duas etapas na sua conta Google
  • Gere uma "Senha de App" em: https://myaccount.google.com/apppasswords
  • Use essa senha ao cadastrar o remetente
2. Fluxo de Uso
  1. Cadastre um remetente com email e senha de app
  2. Use o remetente cadastrado para enviar emails
  3. Consulte o histórico para ver todos os envios
3. Exemplo JavaScript
// Enviar email via JavaScript
fetch('${window.location.origin}/send-email', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    remetente: 'seu-email@gmail.com',
    destinatario: 'destino@exemplo.com',
    subject: 'Teste via API',
    message: 'Este é um teste de envio via API'
  })
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Erro:', error));
