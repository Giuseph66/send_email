API Documentation

Base URL

${window.location.origin}

Enviar Email

POST /send-email 
curl -X POST ${window.location.origin}/send-email \
  -H "Content-Type: application/json" \
  -d '{
    "destinatario": "destino@exemplo.com",
    "subject": "Assunto do email",
    "message": "Conteúdo da mensagem"
  }'

Usuário autenticado

GET /auth/me 
curl ${window.location.origin}/auth/me

Histórico de Emails

GET /emails 
curl ${window.location.origin}/emails

Manual de Uso

1. Configuração do Google
  • Ative a Gmail API no Google Cloud
  • Configure OAuth Client Web e redirect URI
  • Entre com Google para autorizar o envio
2. Fluxo de Uso
  1. Entre ou cadastre sua conta Google
  2. Envie emails pela conta logada
  3. Consulte apenas seu historico
3. Exemplo JavaScript
// Enviar email via JavaScript
fetch('${window.location.origin}/send-email', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    destinatario: 'destino@exemplo.com',
    subject: 'Teste via API',
    message: 'Este é um teste de envio via API'
  })
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Erro:', error));
